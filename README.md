# Bootstrap BOSH Director on AWS

## TL;DR

```
git clone git@github.com:aussielunix/aws_bosh_bootstrap.git
cd aws_bosh_bootstrap
cp .envrc.example .envrc
source .envrc
bbl up
source .envrc
bosh env 
```
